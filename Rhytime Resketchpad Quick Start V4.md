# Rhytime Resketchpad Quick Start

## Workflow
Create new song / import MIDI -> load sounds -> select synth ch -> edit song track timeline / synth presets

Play offline, download it on KVR 
to play anywhere from your usb key, use Edge or Chrome browsers to connect any MIDI input device to Resketchpad.


1. Start with `New song` or `Import MIDI`.
2. Use `Reload sounds` to load a font, sample, or sound setup.
3. Choose the active `Track ch` and `Font ch`.
4. Pick the edit mode you want:
   - `resize LATCH`
   - `shift LINES`
   - `shift FREQ`
   - `pace DRAW`

5. Edit in the timeline, then refine sound with synth presets or soundfont EQ / reverb.

## Song editing
To edit a song bar, the timeline is the place to work. Use the selectors and buttons in the same order you read the song:

Bar / beat view -> zoom -> pan -> select note -> modify start / end / pitch / rest -> hear result

For the "Group"select:

Click Group assign ON or press "G" to toggle Group assign on/off,
next you can drag a box to collect notes, release the mouse when done,
then use Assign selected → current group,
your group owns the notes now.

A few modifiers are already built in:
Alt key while clicking a note  is to add individual notes to the current selected notes
Shift makes the box grow from the center
Ctrl switches the match style to directional/containment behavior
